# breakwall - 推到防火长城
众所周知防火长城是中共为了掩盖真相建立的，随着中国内部越来越复杂，民怨此起彼伏，因此想到一个计划，试图通过一些手段推倒防火长城, 希望各位反对暴政同伴们一起交流技术，为了我们的自由一起努力

1. 通过DDOS攻陷防火长城
2. 通过攻击一些共产宣传工具, 如新华社， 环球时报
3. 使用http不停的试图连接耗尽共军资源
